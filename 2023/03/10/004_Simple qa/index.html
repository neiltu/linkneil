Simple question and answer

spring & springboot 框架

spring是輕量級的Java框架，非侵入式，框架api業務邏輯
spring 容器，透過設定去管理物件生命週期
IOC控制反轉，依賴注入，不必寫很多關聯，需再注入就可以
AOP，記錄安全交易，需要再縫入，跟業務邏輯分開的
springboot將複雜簡化，不用過多繁雜的xml配置，注入即用

Spring RestController & Controller差異?

@RestController = @Controller + @ResponseBody，@Controller表明該類內的所有方法預設返回頁面，@ResponseBody的方法返回資料，寫入HTTP response body

請問Java ORM相關技術為何?

以前使用是Hibernate透過xml才進行，現在使用JPA透過@來進行。

是否使用過AOP技術?

我使用過 AOP 技術。例如在專案中使用 Spring AOP 實現日誌記錄和權限驗證。

請問呼叫API是否有再進行身份驗證?

API 時可以使用身份驗證方式如 OAuth 2.0, JWT token, API key等。例如使用 OAuth 2.0 實現API授權，保證只有授權的用戶才能存取資源。

複習singleton

  1. Singleton 是一種設計模式，它確保在整個程式中只有一個物件實例存在。這樣可以避免多個物件實例之間的競爭和衝突。這種模式常用於創建全域物件，例如單例模式可用於創建單一類別的實例，並確保在整個應用程式中只有一個實例存在，這樣可以確保整個應用程式中的所有部分都使用相同的實例。

資料結構 (HashSet, TreeSet, queue, LinkedList)

  1. HashSet 是 Java 中的一種實現了 Set 介面的類，它使用雜湊表來存儲元素。它不保證元素的順序，並且不允許重複元素。它提供了非常快的查找和插入操作。
  2. TreeSet 是 Java 中的一種實現了 Set 介面的類，它使用紅黑樹來存儲元素。它保證元素的順序，並且不允許重複元素。它提供了較慢的查找和插入操作，但提供了更快的排序操作。
  3. Queue 是 Java 中的一種實現了 Queue 介面的類，它提供了先進先出（FIFO）的數據結構。這種類型的數據結構可以用於實現任務排序，消息傳遞等。
  4. LinkedList 是 Java 中的一種實現了 List 介面的類，它使用鏈結串列來存儲元素。它可以在鏈結串列的開頭和結尾快速插入和刪除元素，並且支援隨機存取。
