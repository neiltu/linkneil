Hexo+Github Pages

Author: Neil Tu Date: March 7, 2023  21:23:00 Category: Hexo

hexo to github pages

Step1 ssh key

Path:C:\Users{user_name}.ssh

Step2 ssh key put on Github

Settings>SSH and GPG keys>New SSH Key>Add SSH Key( id_rsa.pub context copy in )

Step3 github pages repo

Project>Settings>Code and automation>Pages

Step4 Deploy to remote sites

D:\hexo\blog > _config.yml
