Spring boot IOC 範例

Spring Boot 是基於Spring框架的一個輕量級應用程序開發框架，它提供了一種便捷的方式來搭建基於Spring框架的應用程序。在Spring Boot中，採用了Inversion of Control (IoC)容器，通過IoC容器可以將對象的控制權交給Spring框架，讓Spring框架去管理這些對象的生命週期。
IoC容器負責創建和管理對象，並在需要時注入相關的對象，將對象之間的依賴關係解耦，讓應用程序更加靈活和可維護。
