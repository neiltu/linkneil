Curl Command

Author: Neil Tu Date: April 5, 2023  20:58:00 Category: Command

Curl

網路操作，包括HTTP請求和文件上傳下載

curl hostname

curl 查詢IP地址

curl IP

curl hostname or IP+port

curl -L -k

--location 或 -L 自動跟隨 HTTP 重定向，如果服務器返回 3xx 狀態碼，curl 會自動跟隨新的 URL 重新發送 HTTP 請求
--insecure 或 -k 忽略 SSL/TLS 證書驗證，即使證書無效或不受信任，curl 也會繼續執行 HTTP 請求
