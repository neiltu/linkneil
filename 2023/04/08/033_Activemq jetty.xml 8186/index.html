建資料夾 mkdir /server 權限 chmod 777 /server 解壓縮 tar -xzvf /server/apache-activemq-5.15.9-bin.tar.gz cd /server/apache-activemq-5.6.2/bin 權限 chmod 755 activemq 執行 ./activemq start >/tmp/smlog 1. ps -ef | grep activemq：此指令將會列出所有正在執行的程序（包括透過 background 方式執行的程序） 2. nc -lp 61616 &：此指令將會開啟一個 TCP 監聽端口，監聽 61616 port 3. netstat -anp：此指令將會顯示所有正在使用中的連線、開啟的 port 號以及相關程序的資訊，其中 netstat 是一個用於查詢系統 TCP/IP 網路狀態和統計資料的指令 4. netstat -an | grep 61616：此指令將會列出所有與 61616 port 相關的連線和程序資訊 5. netstat -nln | grep 61616：此指令將會列出所有正在監聽 61616 port 的程序資訊 6. web console管理介面: http://localhost:8161/admin/ 1. -a / --all：顯示所有資料，不管是否隱藏或是系統的資料 2. -n / --numeric：顯示 IP 地址和 port 號，不轉換成主機名稱或服務名稱 3. -l / --listening：只顯示正在監聽的連線 4. -p / --processes：顯示每個連線對應的程序名稱 5. -e / --extended：顯示詳細資訊，如 UID 和 GID 6. -f / --full：顯示完整的資訊，包括各種狀態和檔案系統資訊 7. -r / --reverse：反向查找，顯示目標的連線或程序 8. -s / --stats：顯示 TCP/IP 網路統計資訊 9. -t / --tcp：只顯示 TCP 連線 10. -u / --udp：只顯示 UDP 連線 11. -w / --raw：只顯示原始連線，不經過協定解析 12. -x / --unix：只顯示 UNIX 套接字連線 apache-activemq-5.5.0/data/kahadb/ ```
